## 🎯 3 قدم ساده

### 1️⃣ دسترسی به وب‌سایت
```
https://medpromx.github.io/cancer
```

### 2️⃣ قبول Token GitHub (اختیاری)
```
Ctrl+G  ← یا دکمه "تنظیم GitHub"
→ Token درست کنید از:
  https://github.com/settings/tokens/new
→ Scope: repo ✓
→ Copy/Paste token
→ تأیید
```

### 3️⃣ شروع ثبت
```
فرم را پر کنید
↓
Ctrl+Enter یا دکمه ذخیره
↓
✓ ذخیره شد!
↓
فرم خالی → آماده برای بیمار بعدی
```
